# 🚀 QUICK START - Personal Autos Aleixo

## ⚡ Começar em 30 Segundos

### Opção 1: Live Server (Recomendado)
1. Abra VS Code
2. Instale extensão "Live Server" (Ritwick Dey)
3. Right-click em `index.html` → "Open with Live Server"
4. ✅ Pronto! Seu site abre em http://localhost:5500

### Opção 2: Abrir Arquivo Direto
1. Abra `index.html` com seu navegador favorito
2. ✅ Site funciona offline completamente

---

## 📸 Adicionar Imagens

### 3 Passos:
1. Coloque as imagens na pasta `images/`
2. Nomes exatos:
   - `aleixo.jpg` (foto proprietário)
   - `antes1.jpg`, `depois1.jpg` (galeria)
   - `antes2.jpg`, `depois2.jpg` (galeria)
3. Salve e recarregue a página (F5)
